# Dentle TikTok Ad

This Remotion composition creates a 60-second vertical ad for Dentle.

Commands:

```bash
npm run video:studio
npm run video:still
npm run video:render
```

The render command exports:

```text
out/dentle-tiktok-ad.mp4
```

The composition uses the real product screenshots from `public/ad/` and is sized for TikTok/Reels/Shorts at `1080x1920`.
