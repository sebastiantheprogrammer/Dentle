# Content Strategy

## Editorial Thesis

The content should make dental learners feel clever while quietly teaching real clinical reasoning.

Good cases are short, specific, and built around one decisive clue. The explanation should teach why the answer is right and why tempting alternatives are wrong.

## Difficulty Ladder

Beginner:

- classic presentation
- common diagnosis
- clue points clearly to answer

Intermediate:

- common diagnosis with a tempting differential
- requires test interpretation
- uses dental terminology

Advanced:

- subtle presentation
- newer classification terminology
- imaging or pathology nuance
- management decision instead of diagnosis

## Review Workflow

1. Draft case.
2. Check terminology.
3. Add aliases.
4. Add concise explanation.
5. Add a case-specific image that directly supports the diagnosis or decision being tested.
6. Dental reviewer approves.
7. Schedule.
8. After publish, track miss rate and confusing guesses.

## Image Rule

Images are clues, not decoration.

- Use a real radiograph, lesion photo, clinical diagram, or synthetic educational image that matches the finding.
- Do not use generic dental-office, smiling-patient, or random tooth photos for cases.
- For radiology boards, the image should show the actual radiographic pattern when licensing allows it.
- For oral pathology boards, the image should show the lesion pattern being tested.
- If a perfect licensed image is not available, use a clearly labeled diagram or generated synthetic image that still teaches the finding.
